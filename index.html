Two AI tools I built for Innova, before day one.

The role is to go department by department and turn manual, document-heavy work into AI-assisted work. Rather than describe how I'd do that, I picked two departments and built something you can actually click. Both run entirely in your browser - no login, no setup - and both are grounded in Innova's real products.

Field technicians · conversational tool

Field Assist

Problem: techs hunt through PDF manuals for the right error code on site.

A troubleshooting copilot. A technician types a fault in plain language and gets a clear, step-by-step fix with the exact manual page it came from - grounded in Innova's real 2.0 error codes.

Open Field Assist
Documentation · ops dashboard

DocSync

Problem: a spec changes and old figures quietly stay in a French datasheet.

A consistency dashboard for a four-country manufacturer. When a spec changes, it finds every manual and datasheet that references it, in every language, and drafts the fix before a stale doc ships.
